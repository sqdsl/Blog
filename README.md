# Blog
基于SSM实现的个人博客
# 关于项目

该项目是基于SSM（Spring+SpringMVC+Mybatis）做的一个博客系统。
主要涉及的包括 JSP，MySQL，Druid连接池，Spring，SpringMVC，MyBatis 等。
前端采用Layui框架和网上搜索的前台样式，通过Maven管理依赖。 <br/>
详细介绍：[https://liuyanzhao.com/6347.html](https://liuyanzhao.com/6347.html)
